- ✅ ترجمة صوت ← نص ← صوت (اختيار الصوت)
- ✅ تعرف تلقائي على اللغة
- ✅ 20 لغة مدعومة
- ✅ اختيار الأصوات المختلفة
- ✅ نافذة عائمة (overlay) فوق التطبيقات 🫧
- ✅ يشتغل في الخلفية مع زر هشوم 🪶
- ✅ وضع المحادثة الثنائية (أنت + الطرف الثاني)
- ✅ ترجمة الفيديوهات
- ✅ تدقيق الكلام
- ✅ نظام تحديثات من GitHub + لوحة تحكم
- ✅ لوحة تحكم المطور (API Key)
- ✅ وضع المطور بكود سري 🔐
- ✅ تصميم احترافي غامق
